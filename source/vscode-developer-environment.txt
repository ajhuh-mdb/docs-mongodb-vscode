Starting in version v0.11.1 the MongoDB Playgrounds develop experience
is greatly enhanced. This page outlines the changes to the developer 
experience and improvements when working in MongoDB Playgrounds.

Enhanced MongoDB query API intellisense
---------------------------------------

Previous versions of MongoDB Playgrounds had inillisense that would autocomplete
based on the JavaScript language. MongoDB Playgrounds use the MongoDB Query API
as the bassis for intellisense syntax suggestions.

This change can help you with better and more tailored suggestions from
the intellisense engine.
